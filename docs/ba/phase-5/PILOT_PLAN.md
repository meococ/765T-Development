## Recommended pilot shape

- 1–2 BIM teams
- 1 BIM Manager / Coordinator champion
- 2–4 BIMer/Drafter users
- duration: 2–4 weeks

## Pilot goals

1. validate project-understanding value
2. validate review/QC usefulness and trust
3. validate preview/confirm low-risk action lane

## Pilot success criteria

- users voluntarily return weekly
- review findings are considered actionable
- preview/confirm lane is used, not ignored
- top assumptions can be re-scored with real evidence
