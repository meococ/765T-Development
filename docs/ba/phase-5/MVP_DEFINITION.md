## MVP includes

1. **Project understanding**
   - bootstrap context
   - deep scan / context summary
   - evidence-backed project brief

2. **Review / QC**
   - smart QC and related inspection
   - evidence-backed findings
   - manager-friendly review surface

3. **Controlled low-risk action**
   - bounded query/filter/highlight and selected safe actions
   - preview -> confirm -> verify pattern

## MVP excludes

- arbitrary code execution in Revit
- in-Revit multi-agent parallel scan
- broad connector/platform ecosystem
- marketplace/community scripting strategy
- proactive nudging by default
