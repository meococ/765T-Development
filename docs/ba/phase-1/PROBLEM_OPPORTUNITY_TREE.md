## Root outcome

Help Revit delivery teams save time **without sacrificing trust, safety, or auditability**.

## Ranked opportunity tree

| Rank | Problem | Current workaround | Opportunity | BA decision |
|------|---------|-------------------|-------------|-------------|
| 1 | Team takes too long to understand a live project/model | manual view hunting, spreadsheet notes, tribal memory | project understanding and context bundle | MVP Slice 1 |
| 2 | BIM Manager spends too much effort on repetitive review/QC before issue | manual spot-checking, fragmented schedules, warnings review | evidence-backed review/QC lane | MVP Slice 2 |
| 3 | Team wants automation but does not trust black-box mutation | manual edits or brittle scripts | controlled low-risk action with preview | MVP Slice 3 |
| 4 | Team wants broad external integrations | one-off scripts, manual export/import | connector strategy | later |
| 5 | Team wants AI to proactively interrupt and coach | human checking, ad hoc QA | proactive suggestion | later / pilot candidate |
| 6 | Team wants AI to write code and execute arbitrarily | custom scripting | dynamic code gen | out of MVP |
