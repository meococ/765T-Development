# BA Archive Policy

| Field | Value |
|-------|-------|
| **Purpose** | Hold superseded or duplicate BA drafts so they are preserved without staying in the main read path. |
| **Inputs** | Older BA drafts when/if they are replaced. |
| **Outputs** | Archived BA history outside the canonical execution pack. |
| **Status** | Active archive policy; currently no superseded BA draft has been moved here yet. |
| **Owner** | Product + Engineering |
| **Source refs** | `docs/ba/README.md`, `docs/INDEX.md`. |
| **Last updated** | 2026-03-24 |

This folder is reserved for **superseded BA drafts or duplicate BA artifacts**.

Rules:
- archive-first, not hard delete
- archived files must not appear in main BA read order
- archived files must not be presented as current BA truth
- if a BA file is superseded later, move it here and keep a short redirect stub only if needed
