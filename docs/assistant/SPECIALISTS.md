# Internal Specialists

Specialists are internal roles that route implementation/review. They are not tied to a specific vendor AI.

## Core Specialists

- `orchestrator` — route mission + compose plan
- `external-broker` — mediate AI/client outside via WorkerHost gateway
- `internal-assistant` — repo-native assistant execution hygiene

- Specialists are only a routing/behavior layer
- Revit mutation must still go through the tool/workflow safety pipeline
