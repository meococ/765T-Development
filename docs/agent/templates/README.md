# Template pack ? handoff and task scaffolds

This folder keeps lightweight templates used in the historical agent-ops lane.
Use them as scaffolds when a task needs explicit ownership, evidence, and clean handoff.

## Templates
- `HANDOFF.template.md` ? handoff skeleton for what changed, evidence, risks, and next step
- `TASK_CARD.template.md` ? task-card skeleton for objective, constraints, acceptance, and disallowed edits
- `firm-overlay/README.md` ? notes for firm/project-specific overlay adaptation
