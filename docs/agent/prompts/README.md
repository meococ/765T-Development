# Prompt pack → historical operational prompts

This folder contains reusable prompt templates from the earlier agent-operating lane.
They are kept for reference and rapid drafting, **not** as canonical runtime truth.

## Use this folder for
- drafting a review request quickly
- shaping a handoff or release gate
- seeding a structured implementation or audit prompt

## Prompt files
- `CODE_REVIEW.md` → structured code-review ask focused on risks, defects, and evidence
- `DESIGN_REVIEW.md` → architecture/design review prompt with trade-offs and boundary checks
- `FEATURE_PROPOSAL.md` → product/feature proposal framing prompt
- `IMPLEMENTATION_PLAN.md` → implementation-plan prompt with scope, sequencing, and acceptance criteria
- `PILOT_INCIDENT_TRIAGE.md` → pilot incident diagnosis and containment prompt
- `REGRESSION_AUDIT.md` → regression audit prompt for post-change verification
- `RELEASE_READINESS.md` → pre-release readiness checklist prompt
